# Brand Design

You are applying the Groundwork AI brand identity to visual materials.

## When to Use This Skill

Use this skill when creating branded assets, reviewing materials for brand consistency, designing pitch decks, creating social media graphics, or evaluating whether a design aligns with brand guidelines.

## Brand Identity

**Visual positioning:** Grounded, warm, outcome-focused — not sci-fi AI, not generic enterprise.

**Color palette:**
- Deep Forest `#1B3A2E` — primary, used for headers, backgrounds, and primary UI elements
- Amber `#D97706` — accent, used for CTAs, highlights, and emphasis
- Parchment `#FAFAF8` — surface, used for backgrounds and content areas
- Use Deep Forest at 10-20% opacity for subtle backgrounds and borders

**Typography:**
- Sohne — display headings (H1, H2)
- Inter — body text, UI labels, form elements
- JetBrains Mono — code samples, technical content, metrics

**Logo:** Root Node — circle with downward radiating lines (network + roots). Use the full logo in headers and hero sections. Use the icon mark alone when space is constrained.

**Imagery style:** Warm, natural light. Avoid stock photos of people pointing at screens. Prefer abstract patterns, data visualizations, and workflow diagrams that feel organic rather than clinical.

## Brand Application Rules

- Never modify the logo colors, proportions, or orientation.
- Maintain minimum clear space around the logo equal to the height of the icon mark.
- Use the Deep Forest + Parchment combination as the default. Reserve Amber for elements that need attention.
- All text must meet WCAG 2.1 AA contrast requirements against its background.
- Avoid using more than 3 colors from the palette on a single asset.
